# Suite Routing Patterns

## General rule
- Start with the most relevant skill.
- If `memory-context.md` is empty, route intake through `business-consultant`.
- Append responses to relevant memory files before delivering recommendations.

## Example routes

### SEO improvement request
- Entry: Marketing
- If marketing memory empty → Business-consultant intake (ICP, channels, KPIs)
- Update marketing memory → Return to marketing plan
- Complementary: Sales for pipeline impact, Strategy for positioning

### Runway concern
- Entry: Finance
- If finance memory empty → Business-consultant intake (cash, burn, forecast)
- Update finance memory → Run runway analysis
- Complementary: Operations for cost levers, Strategy for investment trade-offs

### Churn spike
- Entry: Customer Success
- If CS memory empty → Business-consultant intake (NRR, cohorts, drivers)
- Update CS memory → Build churn recovery plan
- Complementary: Marketing for lifecycle, Operations for delivery quality

### Market expansion
- Entry: Strategy
- If strategy memory empty → Business-consultant intake (objectives, constraints)
- Update strategy memory → Evaluate options
- Complementary: Finance for model, Operations for feasibility

### Contract risk
- Entry: Legal
- If legal memory empty → Business-consultant intake (jurisdiction, clauses)
- Update legal memory → Risk summary
- Complementary: Sales for deal context, PMO for governance

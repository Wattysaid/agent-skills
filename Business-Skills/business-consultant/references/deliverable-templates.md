# Deliverable Templates

## One-page executive summary
- Objective
- Key findings (3-5 bullets)
- Recommendation
- Impact and risks
- Next steps

## Diagnostic memo
- Background and scope
- Data reviewed and gaps
- Observations
- Root causes (if known)
- Options and trade-offs

## Strategy options table
- Option
- Expected impact
- Cost/effort
- Time to value
- Risks
- Dependencies

## KPI outline
- Metric name
- Definition
- Data source
- Cadence
- Owner

## Implementation plan
- Workstreams
- Milestones
- Owners
- Dependencies
- Risks and mitigations

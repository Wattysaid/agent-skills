# Component Index

Use this index to choose the most appropriate template. All components live under `assets/components` and are Astro components. Use Vue islands from `assets/vue` when interactivity is required.

## Compound Components

- `Accordion.astro` + `AccordionItem.astro` + `AccordionTrigger.astro` + `AccordionContent.astro`
- `Banner.astro` + `BannerContent.astro` + `BannerTitle.astro` + `BannerDescription.astro`
- `SidebarMenu.astro` + `SidebarMenuItem.astro` + `SidebarMenuButton.astro`

## Core UI

- `Button.astro` - Primary action and links.
- `Card.astro` - Standard content container.
- `Container.astro` - Layout wrapper for width and padding.
- `Section.astro` - Page section with optional headings.
- `Badge.astro` - Small status tag.
- `Avatar.astro` - User/profile avatar placeholder.

## Navigation

- `Header.astro` - Page header.
- `Footer.astro` - Page footer.
- `Sidebar.astro` - Sidebar layout wrapper.
- `Tabs.astro` - Tabs layout shell (pair with `Tabs.vue`).

## Content Blocks

- `HeroSection.astro` - Page hero sections.
- `CallToActionSection.astro` - CTA blocks.
- `FeaturedResourcesSection.astro` - Resource listings.
- `BlogPostsSection.astro` - Blog listing blocks.
- `QuickStatsSection.astro` - Metrics blocks.

## Data/Forms

- `Form.astro` - Form shell.
- `Input.astro` - Form input shell.
- `Select.astro` - Select shell (pair with `Select.vue`).
- `Textarea.astro` - Textarea shell.

## Tables/Lists

- `AuditTable.astro` - Audit table layout.
- `Table.astro` - Generic table layout.

## Charts/Media

- `Chart.astro` - Chart container shell.
- `Carousel.astro` - Carousel shell (pair with `Carousel.vue`).
- `RoiChart.astro` - ROI chart shell.

## Utilities

- `Alert.astro` - Alert shell (pair with `Dialog.vue` if needed).
- `Dialog.astro` - Dialog shell (pair with `Dialog.vue`).
- `DropdownMenu.astro` - Menu shell (pair with `DropdownMenu.vue`).
- `Popover.astro` - Popover shell (pair with `Popover.vue`).
- `Tooltip.astro` - Tooltip shell (pair with `Tooltip.vue`).
- `Toast.astro` / `Toaster.astro` - Toast shells (pair with `Toast.vue`).
- `Sheet.astro` - Sheet shell (pair with `Sheet.vue`).

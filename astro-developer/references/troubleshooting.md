# Troubleshooting and Tips

- Ensure your Node.js version is supported (v18.17.1 or v20.3.0).
- Use the development server's error overlay to debug template or script errors.
- For live reloading issues, restart the dev server (`Ctrl+C` then `npm run dev`).
- When using TypeScript, keep `extends: "astro/tsconfigs/strict"` to benefit from recommended compiler settings.
- Consult the Astro documentation at https://docs.astro.build for reference on routing, configuration, and API endpoints.

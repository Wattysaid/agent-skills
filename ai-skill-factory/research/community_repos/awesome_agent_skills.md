# Awesome Agent Skills (Community Repositories)

## Purpose
Curated collections of skills and agent extensions that work across multiple AI coding agents.

## Common Themes
- SKILL.md instruction format
- Folder-based discovery
- Tool wrappers in Python or Node
- YAML/JSON manifests

## Typical Categories
- DevOps
- Data Science
- Web scaffolding
- Documentation generation
- Process mining and BPMN

## How to Use These Repos
1. Browse skill folders
2. Copy into your agent's skills directory
3. Adjust trigger rules and commands
4. Version and test

## Lessons Learned
- Smaller skills compose better
- Explicit triggers outperform implicit heuristics
- Scripts outperform long instruction blocks

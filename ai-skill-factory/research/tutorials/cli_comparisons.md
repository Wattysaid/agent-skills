# CLI Coding Agents Comparison

## Codex CLI
Strengths:
- Lightweight
- Strong file manipulation
- Skills support

Weaknesses:
- Less built-in project context management

## Claude Code
Strengths:
- Excellent long-context reasoning
- Strong project understanding
- CLAUDE.md conventions

Weaknesses:
- Heavier runtime

## Gemini CLI
Strengths:
- Free tiers
- Plugin ecosystem
- Tight Google Cloud integration

Weaknesses:
- Skills not yet standardised

## Implication for Skills Design
Design skills to be:
- File-system based
- Instruction-driven
- Tool-agnostic

Then adapt invocation per CLI.
